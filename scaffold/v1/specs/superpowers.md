# Git Superpowers System Specification

The Git Superpowers system provides high-leverage Git commands designed for high-velocity development and seamless agent-human collaboration.

## Core Superpowers

1. **Spork (/spork)**: One-command branching and remote syncing.
2. **Strail (/strail)**: Compact, visual history trail of the recent work context.
3. **Spdiff (/spdiff)**: Semantic, structural diff that ignores generated files and noisy formatting.
4. **Spwipe (/spwipe)**: Safe context reset by stashing current work before cleaning.
5. **Spmerge (/spmerge)**: Smart merging with automated conflict resolution analysis.

## Components

- **Git Superpower Agent**: Resolves merge conflicts and analyzes work history.
- **Superpowers Extension**: Implements the command suite and TUI visualizations.
